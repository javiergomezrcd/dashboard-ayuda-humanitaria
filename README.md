# Dashboard de ayuda humanitaria — Terremoto Venezuela 2026

Buscador en vivo de pacientes ingresados (datos del registro oficial) y cifras agregadas,
para ayudar a las familias a localizar a sus seres queridos tras el terremoto del 24-jun-2026.

## Privacidad
- Solo muestra: nombre, hospital, fecha de ingreso e identificador interno.
- **No** publica teléfonos, cédulas, direcciones ni coincidencias individuales sin verificar.
- Sin cookies ni rastreadores. Consulta de solo lectura con clave anónima.

> Datos anónimos y agregados. Para verificar coincidencias, contacte con las autoridades
> competentes. No comparta información personal sin consentimiento.
